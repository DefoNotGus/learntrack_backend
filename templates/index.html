Welcome to the LearnTrack API

This API provides personalized content recommendations based on user topics and available content.

How to Use

To interact with the API, send a POST request to the /recommend endpoint with the following JSON payload:

{
    "user_topics": ["machine learning", "data science"],
    "content": [
        {"category": "AI", "title": "Introduction to AI", "description": "Learn the basics of AI"},
        {"category": "ML", "title": "Deep Learning Basics", "description": "An intro to deep learning"}
    ]
}

The API will return the top 5 recommended knowledge cards based on the provided user topics.

Test the Connection

To check the connection, visit the JSON welcome endpoint.
